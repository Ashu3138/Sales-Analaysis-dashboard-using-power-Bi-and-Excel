# Sales-Analaysis-dashboard-using-power-Bi-and-Excel
This project presents an interactive E-Commerce Sales Dashboard built using Power BI to analyze sales performance across multiple dimensions such as time, region, category, product, and customer segment. The dashboard enables data-driven decision-making by providing clear insights into revenue trends, profitability, and operational efficiency.

## Project Objectives

- Analyze **Year-to-Date (YTD)** Sales, Profit, Quantity, and Profit Margin  
- Compare **current year vs previous year** performance  
- Identify **top and bottom performing products**  
- Understand sales distribution by **region and category**  
- Enable **data-driven business decisions**

---

##  Key Metrics (KPIs)

- YTD Sales  
- YTD Profit  
- YTD Quantity  
- YTD Profit Margin (%)  
- Year-over-Year (YoY) Growth  

---

##  Dashboard Features

- Segment filter: **Consumer | Corporate | Home Office**  
- Sales by Category with **YoY comparison**  
- **Top 5 & Bottom 5 products** by YTD Sales  
- **Region-wise sales analysis**  
- **Interactive map visualization**  
- **Shipping mode performance analysis**  
- Dynamic slicers and **drill-down functionality**



---

##  Tools & Technologies

- **Power BI Desktop**  
- **DAX** (YTD, YoY, Profit Margin calculations)  
- **Power Query** (Data cleaning and transformation)  
- **Excel / CSV** as data source  

---

##  Data Modeling

- Implemented **star schema** for optimized performance  
- Created **calculated measures** using DAX  
- Applied **conditional formatting** to highlight trends  

---

##  Business Insights

- Identified **high-performing regions and categories**  
- Highlighted **products contributing to low profitability**  
- Analyzed the **impact of shipping modes on sales**  
- Enabled continuous **monitoring of sales trends over time**
